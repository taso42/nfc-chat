Overview

ajdigital wrote the first flash chat client for NFC. You can find more information on www.ajdigital.com/nfcchat and also in the file /web/flash/index.html which is included with NFC. There is also a support forum for the flash client on sourceforge.

Running the flash client

The flash client can be found in the directory /web/flash. If you are running NFC in servlet mode you can access the client through http://<your_server>/nfc/flash/nfc.html. You may have to edit the file nfc.html to set the appropriate values for _host and _port.
  • _host: the host where NFC is running, e.g. www.your_domain.com
  • _port: the port on which NFC is running. NFC's default is 7777 as you probably know.
When editing nfc.html remeber to change the parameters in both the <PARAM> and <EMBED> tags.
